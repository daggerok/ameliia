How to Play:

  • 🖱️ Left Click - Place block
  • 🖱️ Right Click - Remove block
  • 🎨 Select blocks below
  • ⌨️ 1-6 - Quick select block type
  • 🖱️ Mouse Drag - Rotate camera
  • 🖱️ Scroll - Zoom in/out
